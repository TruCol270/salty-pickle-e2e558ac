

## Salty Pickle — Landing Page

A single-page landing experience with 90s grunge/hip-hop energy, inspired by Tony Hawk Pro Skater and Liquid Death aesthetics.

### Visual Direction
- **Gritty, high-contrast palette**: Black/dark charcoal background, neon greens and yellows (Liquid Death energy), splashes of hot pink or orange
- **Typography**: Bold, distressed/scratchy display fonts for headings (Google Fonts like "Bungee Shade", "Rubik Dirt", or "Metal Mania"), paired with a clean sans-serif for body
- **Textures & Details**: Grain/noise overlays, torn paper edges, spray paint splatters, skate deck graphic energy
- **Illustrations/Graphics**: Skull motifs, dripping text effects, bold angular shapes — everything screams counterculture
- **Animations**: Subtle glitch effects, elements that feel raw and alive

### Page Structure

**1. Hero Section**
- Massive bold headline: something like "TRAIN HARD. RUN DIRTY." with distressed text styling
- Subheadline explaining the adaptive AI training plan concept
- Prominent "GET STARTED" CTA button with aggressive styling (neon border, hover effects)
- Background with gritty texture/pattern

**2. Features Section (3-4 cards)**
- Adaptive plan adjustments based on performance
- AI-powered scheduling that learns your body
- Weather/fatigue-aware recommendations  
- Progress tracking with attitude
- Each card with bold iconography and short punchy copy

**3. Footer CTA + Sign Up**
- Final bold call to action
- Sign up / get started button
- Minimal footer with links

### Technical Approach
- All built in React with Tailwind CSS
- Custom CSS for grunge textures, glitch animations, and distressed effects
- Google Fonts for the display typography
- Fully responsive (mobile-first)
- GitHub repo integration deferred to a follow-up phase
